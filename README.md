## Vesper Obsidian Nexus（全栈版）

- 提供 Node.js + Express + SQLite 的后端，支持账号注册/登录、验证码验证、开发者识别码绑定、批量生成/收回识别码、优惠码生成、VIP 订单记录与自动确认、审计日志、图标与知识块的持久化管理。
- 前端统一迁移到 `public/index.html`，保持纯黑、干净的美术风格，支持账户流程、开发者绑定、图标分组管理、知识/工具卡片、VIP 提交、用户与审计查看。
- 默认最高权限识别码为 `Vesper`，迁移脚本自动注入，可生成同级别 Root/Admin/Developer 识别码。
- 数据库存储在 `./data/app.db`，API 基于 JWT 保护，接口定义详见 `src/server.js`。

### 本地快速启动
1. 安装 Node.js（18+）。
2. 执行 `npm install`（若网络受限，可改用镜像，必要时手动下载依赖）。
3. 初始化数据库：`npm run migrate`。
4. 启动服务：`npm start`，浏览器访问 `http://localhost:3000`。

### 部署与域名绑定
- 参考 `DEPLOY.md` 获取从 Windows 首次部署到 Linux/Nginx/域名绑定、反向代理、HTTPS 的逐步操作与排障。
- 生产环境请在环境变量中设置 `JWT_SECRET`、`DATABASE_PATH`，并在支付网关回调时调用 `/api/vip/confirm` 完成自动升级。
